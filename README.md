# Streamlit - Pokedex

Proyecto de una Pokédex interactiva desarrollada con Streamlit.

Esta aplicación permite buscar y visualizar información detallada de Pokémon de manera rápida, organizada y visualmente atractiva. Cuenta con una interfaz intuitiva y moderna, pensada para facilitar la exploración de datos, estadísticas y características de cada criatura.

Características principales:

- Búsqueda rápida por nombre o número de Pokédex.
- Visualización clara de estadísticas y tipos.
- Diseño simple y elegante, ideal para presentaciones.
- Ejemplo práctico de aplicación web en Python.

Perfecto para quienes quieren aprender a crear interfaces interactivas con Streamlit y explorar los datos de la franquicia Pokémon de forma entretenida y profesional.
